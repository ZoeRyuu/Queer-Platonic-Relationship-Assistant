# QPR 助手 — 新增需求

> 本文档为增量需求,基于已有的 MVP(自我探索问卷 + 关系画像 + 小百科)。
> 包含两项变更: ① 性话题纳入自我探索问卷; ② 新增独立模块"我的光谱"。

---

## 变更一: 自我探索问卷新增"性与身体边界"章节

归属: 第一模块"自我探索",作为问卷的 **章节 E**,与章节 A–D 并列。

### 定位与安全设计
- **可选章节**: 章节列表中标注"可选"。入口页显示提示: "本章节涉及性相关话题。你可以完全跳过,跳过不影响关系画像的生成。"
- 每道题均提供 **"不想定义 / 暂不回答"** 选项,该选项视为有效答案(画像中如实呈现),而非未完成。
- 文案使用临床中性的准确术语,不色情化、不病理化。
- 本章节答案与其他章节相同,仅存本地 IndexedDB;导出画像时用户可单独勾选是否包含本章节内容。

### 题目设计

**E1. 意愿梯度**(每项统一选项: 愿意 / 视情况协商 / 不愿意 / 还不确定 / 不想定义)
- 出于性欲的肢体接触(带有性意味的抚摸、亲吻等)
- 边缘性行为(非纳入式的性接触)
- 纳入式性行为

**E2. 性在关系中的位置**
- 单选: 希望关系包含性 / 明确不包含 / 开放协商 / 不预设 / 不想定义

**E3. 性的排他性**
- 单选: 要求排他 / 不要求排他 / 需要协商 / 不适用 / 不想定义

**E4. 性话题的沟通**
- 谈论性话题时我的舒适度(滑块: 非常不适 ←→ 完全自在,可跳过)
- 我偏好的沟通方式(多选: 当面直说 / 书面表达 / 借助工具或清单 / 希望对方先开口 / 自由文本)

### 与其他部分的联动
- 意愿题与任何性取向标签**互相独立**,应用不做任何推断(例: 不因用户在光谱中靠近"无性恋"就隐藏或预填本章节)。
- E1–E3 的结果在"关系画像"中以独立分区展示,标题为"性与身体边界"。

## 变更二: 新增独立模块"我的光谱"

归属: 与"自我探索""小百科"并列的 **新一级模块**,首页增加入口。

### 功能描述
多维光谱自我定位工具。用户在每条轴上拖动滑块,标记自己此刻在光谱上的位置。

### 七条光谱轴

| 维度 | 左端 | 中间参考点 | 右端 |
|---|---|---|---|
| 性别认同 | 男性 | 非二元 | 女性 |
| 性别表达强度 | 高度男性化 | 雌雄同体 | 高度女性化 |
| 性取向 | 异性恋 | 双 / 泛性恋 | 同性恋 |
| 受性吸引强度 | 无性恋 | 灰性恋 | 有性恋 |
| 性欲望 / 性冲动 | 低欲望 | 中等 | 高欲望 |
| 浪漫倾向 | 无浪漫倾向 | 感兴趣 | 强烈浪漫倾向 |
| 关系结构偏好 | 单偶 | 开放 | 多偶 |

### 交互要求
- 每轴为连续滑块(内部值 0–100),**默认无初始位置**——用户未触碰前滑块不显示在轴上,避免暗示"默认/正常值"。
- 每轴可单独标记 **"跳过此轴"**。
- 轴端点与中间参考点仅为坐标参考,页面说明文案明确: 这是自我探索工具而非测试,没有"正确"位置,两端不代表对立或优劣。
- 每个维度标题可点击,跳转小百科对应词条(小百科需补充: 非二元、灰性恋、单偶/多偶等词条)。

### 快照功能
- 用户可将当前七轴状态保存为一份 **快照**(自动记录日期,可加备注)。
- 快照列表按时间排序;支持选择 2–3 份快照 **叠加对比**(同一轴上以不同颜色/透明度显示不同时期的位置),呈现光谱随时间的流动。
- 快照可删除;删除需确认。

### 展示与导出
- 最新光谱并入"关系画像"页,作为独立分区;导出画像时可勾选是否包含。
- 支持将光谱单独导出为图片(样式参考坐标轴光谱图: 每行一条横轴,标记点落在轴上)。

### 数据结构建议
```ts
interface SpectrumSnapshot {
  id: string;
  createdAt: string;      // ISO 日期
  note?: string;
  axes: {
    axisId: string;       // 'gender-identity' | 'gender-intensity' | ...
    value: number | null; // 0–100,null 表示跳过此轴
  }[];
}
```

## 页面结构变更

```
├── 首页(新增"我的光谱"入口)
├── 自我探索
│   ├── 章节列表(A–D + 章节E,E标注"可选")
│   └── 我的关系画像(新增分区: 性与身体边界、我的光谱)
├── 我的光谱 ★新增
│   ├── 光谱编辑页(七轴滑块)
│   └── 快照列表 + 对比视图
├── 小百科(补充光谱相关词条)
└── 设置(不变)
```

## 开发顺序建议(增量)

1. 问卷新增章节 E: 可选章节入口提示、跳过逻辑、"不想定义"选项、题目组件
2. 关系画像新增"性与身体边界"分区,导出勾选逻辑
3. "我的光谱"模块: 七轴滑块组件(无默认值 + 跳过此轴)
4. 快照的保存 / 列表 / 对比视图
5. 光谱并入关系画像 + 单独导出为图片
6. 小百科补充词条,维度标题跳转链接
